# 薬剤比率計算機 📱

モバイルフレンドリーな薬剤の比率計算アプリケーション

## 主な機能

### 🧮 比率計算
- 全体の重さと比率(1剤:2剤)を入力することで、各薬剤の必要量を自動計算
- 小数点以下1桁まで対応
- リアルタイムで計算結果を表示

### 📱 モバイルフレンドリー設計
- **大きなタップ可能ボタン** - モバイルでの操作性を最優先
- **数字パッドUI** - 計算機風の直感的な入力インターフェース
- **レスポンシブデザイン** - スマホ・タブレット・PCすべてに対応
- **タッチ操作最適化** - タップ時のフィードバック、スムーズなアニメーション

### 💾 データ管理機能
- **計算履歴** - 最新20件の計算結果を自動保存（localStorage使用）
  - 履歴から過去の計算を再読み込み可能
  - 個別削除・全削除対応

- **プリセット機能** - よく使う比率を名前をつけて保存
  - カスタム比率の保存
  - ワンタップで読み込み
  - 自由に編集・削除可能

- **メモ機能** - 作業メモや備考を自由に記録
  - 日時付きで自動保存
  - メモの追加・削除が簡単

### ⚡ 使いやすい入力方法
- **入力フィールド選択式** - 入力したい項目をタップして選択
- **カスタム数字パッド** - 大きなボタンで快適に数値入力
- **+1/-1ボタン** - 微調整も簡単
- **クリアボタン** - 入力のリセットがワンタップ

### 🎨 モダンなデザイン
- マテリアルデザイン風のクリーンなUI
- 見やすいカラースキーム（プライマリカラー: ティール）
- 視認性の高い計算結果表示
- アニメーション付きのスムーズな操作感

## 技術スタック

- **React 17** - UIライブラリ
- **SCSS** - スタイリング（CSS Variables使用）
- **localStorage** - データ永続化

## 使い方

### インストールと起動

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm start

# 本番ビルド
npm run build
```

### 基本的な使い方

1. **計算タブ**で計算を実行
   - 「全体の重さ」「比率1」「比率2」の入力欄をタップ
   - 数字パッドで値を入力
   - 計算結果が自動で表示されます

2. **履歴タブ**で過去の計算を確認
   - 「履歴に保存」ボタンで保存
   - 履歴から過去の計算を再読み込み可能

3. **プリセットタブ**でよく使う比率を管理
   - 計算画面で比率を入力後、名前をつけて保存
   - 保存したプリセットをワンタップで適用

4. **メモタブ**で作業メモを記録
   - 配合のコツや注意事項などを自由にメモ
